# Financial Data Automation

## Описание

**Financial Data Automation** — проект по автоматизации обработки финансовых транзакций.

Основная задача проекта — объединить процессы загрузки данных, нормализации, проверки, хранения и подготовки информации для дальнейшей аналитики.

Проект разрабатывается как единая ETL-платформа, позволяющая автоматизировать обработку финансовых данных из различных источников.

---

## Основные возможности

* автоматическая загрузка данных;
* обработка отчётов различных платёжных систем;
* нормализация финансовых транзакций;
* очистка и валидация данных;
* подготовка данных для аналитики;
* работа с SQL-запросами и представлениями (Views);
* автоматизация повторяющихся процессов;
* ведение журналов выполнения (логирование).
